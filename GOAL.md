# Goal

Build and operate a safe Gmail cleanup system for a mailbox that was unmanaged for more than seven years.

Immediate goal:

1. Double/triple-check every message moved or planned for Trash before permanently deleting anything.
2. Use local Qwen/llama.cpp review as an additional cautious reviewer.
3. Restore or label possible false positives, especially immigration/legal, studies, real attachments, finance/security, receipts/orders, and human conversations.
4. Keep generated Gmail data local and private.
5. Keep the workflow resumable and easy to restart after long overnight runs.

Longer-term goal:

- Keep Gmail clean with periodic maintenance scans instead of another multi-year backlog.
